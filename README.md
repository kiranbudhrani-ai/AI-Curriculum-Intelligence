# AI Growth Signals in the Curriculum

This repository is intended to host the AI curriculum growth dashboard package as a GitHub Pages site.

## Files to upload

Place the following HTML files in the repository:

- `index.html`
- `resources/full_dashboard_v12.html`
- `resources/dashboard_1_executive_summary.html`
- `resources/dashboard_2_interactive_explorer.html`
- `resources/dashboard_3_college_scorecards.html`
- `resources/dashboard_4_one_page_printable.html`

## GitHub Pages setup

After the files are uploaded, go to **Settings > Pages** and select:

- Source: Deploy from a branch
- Branch: main
- Folder: root

Then save. GitHub Pages will publish the site at a URL like:

`https://kiranbudhrani-ai.github.io/AI-Course-Signals/`
